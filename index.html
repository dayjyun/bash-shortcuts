Bash Shrtcuts

Make Programming Easy

As programmers, knowing keyboard shortcuts to implement when coding speeds up productivity and saves time. You are already spending plenty of hours typing code. Instead of constantly switching back and forth between your mouse and keyboard, keep your hands in place and take advantage of the simplicity offered with the shortcuts in-place to make your life easier.


On top of that, creating a few of your personalized commands is an added feature to mastering code. Having your keyboard work the way you want makes programming enjoyable and will cause less errors. Reduce your time from moving away from the keyboard and focus on what's more important in front of your screen.


In order to build your own shortcuts, make sure you have the file necessary to read them. Let's take care of that.

Note. This is intended for command lines running Bash. It may not work in other shell languages.

  1. Open up your CLI
  2. Navigate to your root directory. Type cd
  3. Open the .bash_profile file on VSCode. Type code .bash_profile

    If it's not present, create the file. Type touch .bash_profile

  4. Type your personalized command.

    The code structure is alias [personalized shortcut name]="command to replace"

  5. Save the file and restart your terminal

That's it! Now it's time to feel like a hacker and play around with your newly created commands. You're on your way to faster, more enjoyable programming.


Feel free to copy the code written in the blocks or try coming up with a few of your own!
